What is a div element?

In HTML, a <div> is a container element used to logically group and organize other HTML elements on a web page. The name "div" stands for "division" or "divider." It does not have any inherent meaning or default styling but serves as a flexible and generic container that allows you to apply your own styles and structure. You can use the <div> element to create layout structures, define sections, group related elements together, apply styles, and control the positioning of elements on a webpage. It is a block-level element, meaning it starts on a new line and takes up the full width available by default, but can be styled and positioned using CSS.

Linking CSS to HTML

Getting your HTML to recogise your CSS file

To link a CSS document to HTML, you can use the syntax:
<link rel = "stylesheet" href = "link to your CSS document">

Coding CSS directly in HTML file

if you want to directly put CSS code in HTML, you can use the syntax:
< styles = "background-color: ;color: ;">
An example: <h1 style = "color: rgb(255,255,255);"> hello world! </h1>
this will produce the sentence "hello world!" in white.
Do note that all CSS assignments from lesson 3 - 4 will also work and have the same syntax.

General tips for organising your website

Using <div>:

The div attribute can be used to seperate parts of your website, which can help reduce the messiness of your code. Keep your code tidy and organized. Use proper indentation and formatting to make it more readable. Comment your code to provide explanations when necessary, especially when dealing with complex div structures.

Use CSS classes:

Assign appropriate CSS classes to your div elements to style them easily. CSS classes help identify and select specific elements, making it easier to apply consistent styling and layout across your website.

Block classes:

It might be easier to organise your website if you think of div as 'blocks' that seperate a section of your website from another.
