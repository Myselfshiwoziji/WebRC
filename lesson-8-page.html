JavaScript Functions

Write your code within a function so that it can be executed on call:

function exampleFunction(){
   alert("Hello, World!")
}

The onClick Event

The simplest way to link your JavaScript function to a user input is with the onClick event assigned to buttons. In your HTML, type:

<button onClick="exampleFunction()">Example Button</button>

to create a button connected to your example function. Clicking it should now run your code!

Uses and examples of parameters

Creating reusable and adaptable code:

By using parameters, you can create functions that can work with different data or conditions. This makes your code more modular, reusable, and adaptable to different scenarios.

Here is an example: Suppose you are making a quiz and want to make a hint button (with an id called Hintbutton) to display a hint (with id Hintparagraph), so you type:
 
function buttonclick() {
 for (let i = 0; i < document.querySelectorAll(#Hintbutton).length; i++) {
  if (paragraph.style.display === 'none') {
   paragraph.style.display = 'block';
   paragraph.style.visibility = "visible";
  } else {
   paragraph.style.display = 'none';
   paragraph.style.visibility = "hidden";
  }
 }
}

However, you find that if you click any button in the iframe, all paragraphs with the id paragraph will show.

so instead, using parameters, you can configurate the code accordingly:

const hint_texts = document.getElementsByClassName("hint-text");
function buttonclick(hint_id){
 for (let i = 0; i < hint_texts.length; i++) {
  let hint = hint_texts[i];
  let display_status = hint.style.display;
 if (i == hint_id) {
  if (display_status == 'block'){
   hint.style.display = 'none';
   hint.style.visibility = "hidden";
  }
  else{
   hint.style.display = 'block';
   hint.style.visibility = "visible";
  }
 }
  else{
   hint.style.display = 'none';
   hint.style.visibility = "hidden";
  }
 }
}

In your HTML, you can then configure each button to run the function with a different hint_id:

<button class="hint-button" onclick="buttonclick(0)">Hint</button>

Mathematical Functions in JS

Some people will roll their eyes over this, but it is pretty important so lets get it done with.

Math.floor()

This function floors the value that is in the brackets, which means it takes the lower integer bound of the number
For example, if you have the number 5.3, it would round to 5
Another example, if you have a number 4.9, it would still round to 4

Math.ceil()

This function is the opposite of the floor function. It takes the upper bound of the number inside the bracket (rounding up).
For example, if you have a number 10.1, it would round to 11.
Another example, if you have a number π, it would round to 4.

Math.random()

Math.random() does what you think it does. It returns a random value from 0 to 1.
*Note: This 'randomly' generated function is called a persudo random function, since it is not truly random. If you would like to learn more about how computers generate 'random' numbers, go to this wikipedia page.
