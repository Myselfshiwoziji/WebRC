Animations

Animations in HTML and CSS are techniques used to bring elements on a web page to life, creating movement and visual effects. They allow developers to add dynamic and interactive elements to their websites, enhancing the user experience.

Making animations

Keyframes

Keyframes are static frames where something happens.

To make a keyframe, you insert a @keyframe in CSS,

Linking keyframes together

After you have a sufficient number of keyframes, you can link them up together to display the full animation to showcase off your creativity!

Assigning HTML to animations % examples

Contrary to the name, if you want to assign html attributes, it would be easier to do it in CSS. To do it in CSS, you can:

Duration

assigning the animation-duration attribute to an animation can control its duration. (make sure to specify whether its in miliseconds or seconds)

Timing

This defines how fast an animation is played during the time which it is played. The prompts can include ease, which is a prompt where it goes from slow to fast exponentially (ease-in, ease-out, and ease-in-out), and linear where it goes from slow to fast in a linear manner.

Iterations

Iterations is the number of times an animation is played. using animation-iteration-count, you can control the amount of times its played, from 1 to infinite for continuous looping.
