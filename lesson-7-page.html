What is JSON?

JSON, or JavaScript Object Notation, is a file format designed to store key-value pairs of variables in a human readable text format. Although designed for JavaScript, the data types it stores are common to most modern programming languages, and its text-based format can be easily parsed, making it a valuable format to store and transmit data between programs.

Creating a JSON File

Like with other file types, give your file a name ending in .json to declare it for JSON. Such as:

examplefile.json

What can JSON be used for?

  Storing Data / Data transmittion

JSON is often used to send data between a server and a client as part of web APIs (Application Programming Interfaces). It allows for structured and organized transfer of data, making it ideal for applications that interact with remote servers.

  Backend

This course will not be covering backend in our 9 lessons, but if you stay for lessons 10 - 12. we will talk about backend, nodes, save data, etc. which can help you make an interactable website.

  Databases

In the future, it may be frequent that you may want to set up a database of sorts for user input, for example for conducting surveys, managing data, and other sorts. In these scenarios, it would be wise to use either mongoDB or SQL to set up a database. Both of these languages rely on JSON to store data, for its compactness and reliability.

JSON Syntax

JSON data comes in a form similar to JavaScript Objects or Python Dictionaries: with key-value pairs. Each value (either a number, string, boolean, array, null, or nested key-value pair) is assigned a string name (the key), from which the value can be called.

However, JSON syntax is a lot stricter than Python or JavaScript! Every key must be a string, enclosed in double quotation marks (""), then followed by a colon (:), then the corresponding value. If you wish to have multiple key pairs, they can be seperated by commas, but the final element can never have a comma. Here is an example of a JSON object:

{
 "name": "John Doe",
 "age": 30,
 "email": "johndoe@example.com",
 "address": {
  "street": "123 Main St",
  "city": "Anytown",
  "state": "Anystate",
  "postalCode": "12345"
 },
 "phoneNumbers": [
  {
   "type": "home",
   "number": "555-1234"
  },
  {
   "type": "work",
   "number": "555-5678"
  }
 ]
}

Accessing JSON in JavaScript

Accessing your JSON file in JavaScript is a little more complicated than defining an src or href as before. There are many ways to link a JSON file to your script, one way is with fetch API:

fetch('./level-plan.json')
.then(response => response.json())
.then(data => {
 var objectFromJSON = data;
.})
..catch(error => {
 .console.log('Error:', error);
.});

Parse

You can also link your JSON through parsing, through the syntax JSON.parse, for example, consider the JSON object above is called jsonData:

var obj = JSON.parse(jsonData);
console.log(obj.name); // Output: John Doe
console.log(obj.age); // Output: 30
console.log(obj.address.city); // Output: Anytown
