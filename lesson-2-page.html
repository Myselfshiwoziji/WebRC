What is an attribute?

An attribute in HTML is like a trait you give to it. It provides additional information to the element, such as its size, width, length, max_word_count, or funtionality. For example, the "href" attribute a tag specifies the URL or web address to which the link should link to. The "src" attribute in the image tag specifies the source or location of the image file to be displayed. Similarly, the "class" attribute is used to assign a class or style to an element, allowing CSS (Cascading Style Sheets, which you will be covered in lessons 3 and 4) to apply specific formatting. More examples will be shown in the next panel.

Examples of attributes in HTML:

Class: The class attribute is used to apply one or more class names to an element. Classes are used to select and style elements with CSS or to define behavior with JavaScript.

Id: The id attribute is used to identify an element within a document (which is the file). It is often used for JavaScript or CSS targeting purposes.

Src: The src attribute is used to specify the source URL of external resources, such as images, audio, video, or script files.

Href: The href attribute is used to specify the target URL of links. It is typically used with anchor elements to link to other pages or resources.

Alt: The alt attribute is used to provide alternative text for elements that isn't displayed, such as images. It is used for accessibility purposes and is displayed if the image fails to load/ the user is not supposed to be able to see the image.

Style: The style attribute is used to apply inline CSS styles to an element. It allows for the customization of various visual aspects, such as color, font-size, margin, etc. (you'll learn about the CSS attributes in lessons 3-4)
