What is HTML?

HTML, or Hypertext Markup Language, is a language for web development on the World Wide Web. HTML specifies the content for a web page to your web browser, which will then interpret and render your website based on the file's markup. In this way, HTML acts as the "skeleton" for your website, from which you will eventually build on with CSS and JavaScript.

In order to make an HTML file in Visual Studio Code;
1. Right click the left hand side menu
2. Create a file named anything you want (as long as it ends with .html).

What is an element in HTML?

In HTML, an element refers to a fundamental building block or a component of a webpage. It represents different parts of the content, structure, or functionality of the webpage. Each element is enclosed within tags, which define the purpose and behavior of that element. Examples of HTML elements include headings, paragraphs, images, links, buttons, and many more. Elements can be styled, manipulated, and arranged using CSS (Cascading Style Sheets) and can also have interactive behaviors through JavaScript (we will discuss CSS and Javascript later in the course).

Examples of elements in HTML:

html:

This is the central element for any form of HTML based coding you will ever do. Almost everything made in the HTML script must be put inside the html element, as it defines that everything inside is an HTML Element. You can think of this tag to be similar to a sentence starter and a full stop.
When you use this element, use the syntax:
<html> </html>

head:

This is an incredibly important tag for any sort of HTML document. The head element is an element that acts as a container for all sorts of metadata for the website. This basically means that all the stuff the website requires but isn't typically shown to the user will go in the header tag. For example, defining the language used for the website, the website icon in the tab, the website name displayed in the tab, etc. This element should be used inside the HTML element and not inside the body element.
When you use this element, use the syntax:
<head> <head>

body

This is another incredibly important element for coding using HTML. This element specifies that everything inside of it is most (if not all) visual elements. This means that almost everything inside of the body element will be visible to the user (you will later see that there are elements that control other functions that will not be visible to the user). To use this element, put it right after (and not inside) the head element.
When you use this element, use the syntax:
<body> </body>

Heading:

This is a very simple element, and is basically a element that contains text for your headings in your website. Interestingly, the syntax for this element is quite unique, as you can label your headings in numerical values. For example, you can have a heading 1 and a heading 2 to differenciate between the two. To type in words for the heading to display, type them inside the heading element. When you use this element, use the syntax:
<h1>This is a heading. Also as a note you can change the number "1" to any other whole number to number them like described as long as that number is smaller or equal to 6 </h1>

Paragraph:

This is another element which deals with text. However, unlike the heading paragraphs are much smaller texts that can contain much more letters than a header. Basically, just think of this element like changing between header and normal text in Google Docs. To type in words for the paragraph to display, type them inside the paragraph element. When you use this element, use the syntax:
<p> Hello world, this is a cool little paragraph that contains lots of knowledge </p>

Button:

This is qutie a straightforward element. This element, when used, creates a button that the user can click on. Much like the paragraph and heading elements, you can place the text that you want the button to display inside the button element. When you use this element, use the syntax:
<button>Click me!</button>

Anchor:

The anchor element is a very important element, and you probably will see it in use quite a lot. The anchor element is a type of element in HTML that allows for text put inside of it to be interactable and, much like a button, allows for it to be clicked. The anchor element is typically seen as a way to link two different pages or destinations together. For now, think of this as a hyperlink. When you use this element, use the syntax:
<a>Go to Youtube</a>

Division:

In HTML, a <div> is a container element used to logically group and organize other HTML elements on a web page. It does not have any inherent meaning or default styling but serves as a flexible and generic container that allows you to apply your own styles and structure. You can use the <div> element to create layout structures, define sections, group related elements together, apply styles, and control the positioning of elements on a webpage. When you use this element, use the syntax:
<div>INSERT ANY ELEMENTS HERE</div>

What is an attribute?

An attribute in HTML is like a trait you give to it. It provides additional information to the element, such as its size, width, length, max word count, or funtionality. For example, the "href" attribute specifies the URL or web address to which the hyperlink should link to. The "src" attribute in the image tag specifies the source or location of the image file to be displayed. Similarly, the "class" attribute is used to assign a class to its element, allowing CSS (Cascading Style Sheets, which will be covered in lessons 3 and 4) to apply specific formatting. More examples will be shown in the next panel.

Examples of attributes in HTML:

Class:

The class attribute is used to apply one or more class names to an element. Classes are used to select and style elements with CSS or to define behavior with JavaScript. When you use this attribute, use the syntax:
<button class = "exampleClass"> </class>

Id:

The id attribute is used to identify an element within a document (which is the file). It is often used for JavaScript or CSS targeting purposes. When you use this attribute, use the syntax:
<button id = "exampleId"> </button>

Src:

The src attribute is used to specify the source URL of external resources, such as images, audio, video, or script files. When you use this attribute, use the syntax:
<img src = "example_img.jpeg"> </img>

Href:

The href attribute is used to specify the target URL of links. It is typically used with anchor elements to link to other pages or resources. When you use this attribute, use the syntax:
<a href = "www.google.com"> </a>

Alt:

The alt attribute is used to provide alternative text for elements that isn't displayed, such as images. It is used for accessibility purposes and is displayed if the image fails to load/ the user is not supposed to be able to see the image. When you use this attribute, use the syntax:
<img alt = "example image for students"> </img>

Style:

The style attribute is used to apply inline CSS styles to an element. It allows for the customization of various visual aspects, such as color, font-size, margin, etc. (you'll learn about the CSS attributes in lessons 3-4) When you use this attribute, use the syntax:
<button style = "color: blue; text-align: center"> </button>
