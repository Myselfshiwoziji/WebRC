What are classes?

Classes in Javascript are an easy way to define and create objects with a blueprint for their properties and methods. Much like Json, they can be used to easily store information about an object.

Method

Methods in Javascript are functions that are associated with a class or an object, which represents behavior or actions that objects of the class can perform. When a method is called, it can access and manipulate the data within the object it belongs to, and it can also interact with other objects or perform other operations.

Constructor

A constructor is a special method that is called when an instance of a class is created. It is used to initialize the newly created object and to set initial values for its properties and perform any other setup that may be required.

Making classes

To make a class, you use the syntax: class example {
 constructor(property1, property2) {
  this.property1 = some property 1;
  this.property2 = some property 2;
 }

 method1 {
  method logic here
 }

 method2 {
  method logic here
 }
}

Keep in mind that you can have as many properties and methods as you want. To assign something to a class, use the syntax:
let name_of_object = new example(property1, property2)
but replace name of object with the name of what you want to call it, example to your class name, and the properties.

To call a method, use the syntax:
name_of_object.method1()
and replace the placeholders accordingly.

Example of classes

Say, you want to have a class 'Circle' that you can call, and you want to be able to get its area. You can make a class like this:
class Circle {
 constructor(radius) {
  this.radius = radius;
 }

 getArea() {
  return Math.PI * this.radius * this.radius;
 }

 setRadius(newRadius) {
  this.radius = newRadius;
 }
}
In this example, you can see that the properties are radius, and the methods are getArea() and setRadius(newRadius).


Or, you want to have a counter that you can add or delete values from:
class Counter {
 constructor() {
  this.count = 0;
 }

 increment() {
  this.count++;
 }

 decrement() {
  this.count--;
 }
}

As you can see, calling the increment() function in the class Counter would raise it by 1, while calling decrement() would decrease the counter by 1.

Subclasses

Sub-classes, also known as derived classes or child classes, are classes that inherit properties and methods from another class, known as the superclass, base class, or parent class. When a class inherits from another class, it automatically gains access to all the public and protected members (properties and methods) of the parent class. The sub-class can also add its own methods and properties, as well as override or extend the behavior of the inherited methods. For example, consider this class: class Animal {
 constructor(name) {
  this.name = name;
 }
 speak() {
  console.log(this.name + ' makes a noise.');
 }
}
As seen from the example, the class animal makes a noise whenever it speaks. However, what if we specifically want a dog (which belongs under the class animal) to bark instead? To do that, we can assign a sub-class:
class Dog extends Animal {
 speak() {
  console.log(this.name + ' barks.');
 }
}
As seen from above, you can use extend to make a subclass (in this case dog of animal), which has all the properties of the class animal but also special methods.

Setting up CSS

Much like starting your first HTML file, you can right click on the left hand side menu to create a new file. This file should not contain spaces and must end with .css to declare that it is a CSS file. Once done, return to the HTML document that you want to link the CSS document to and go inside the head element of the file. Once there, type:
<link rel = "stylesheet" href = "The CSS file's name">
Once you are done with that, you should now be able to code your CSS file such so that it should influence the visuals of the HTML page. Check out the next tab for more details regarding how to style a CSS file:

CSS Declarations

CSS Declarations are simple lines of code that help us define exact attributes (or in this case styles) that we want specific elements to take on. To use CSS Declarations, you have to follow the syntax shown below:

.className{
Insert CSS Declarations here
}

Color:

This is a very simple declaration to understand, as this declaration simply changes the color of text within the element to a color of your choice. When choosing a color, you have to use the RGB system to create your desired color. To use this declaration use the syntax shown below
color: rgb(0,0,0);

(Note: when using rgb Visual Studio Code already gives you a large variety of diffeent colors to work with, so you often don't fully need to manually type in the rgb values of the color you want)

Background color:

This is another simple declaration to understand, as it simply changes the background color of the element. Much like the color declaration, you also have to utilise the rgb system to select your colors. When you use this declaration, use the syntax:
background-color: rgb(0,0,0);

Font Size:

This declaration is in charge of what the size of the text will be. To use this, you have to define the font size as a specific size in terms of pixels (we will discuss alternate unit of measurements other than pixels later). When you use this delcaration, use the syntax:
font-size: 16px;

Font Family:

This delcaration is used to change the font of the selected element. To use it, simply just state what font you would like to use (eg: Arial, Times New Roman, etc). However, there are many fonts that are unsupported in CSS, so if you want to import your own fonts that you would like to use, please check out this tutorial. When you use this declaration, use the syntax:
font-family: 'Times New Roman', Times, serif;

Border:

This declaration is used to add borders to your selected elements. When using this declaration, you have to first declare the width of the border (for now use pixels), then declare the style of the border (which can be: solid, dotted, double, dashed, inset, outset, ridge, groove, and hidden), and finally declare a color for the border (once again, you have to use the RGB system to define the color). An example of how the syntax would look like would be:
border: 5px, solid, rgb(0,0,0);

Border radius:

This is a very simple declaration and links back to the previous declaration on borders. The border radius declaration simply declares how round the edges of the border will become. To use this, you have to input how round you want to make the borders using pixels (or other supported units of measurement in Visual Studio Code).
Note: if you want to make a completley circular border, you can set the border radius to the height of the element. When you use this delcaration, use the syntax:
border-radius: 2px;

Padding, Border and Margin

Padding

In HTML, padding refers to the space between the content of an element and its border. It allows you to create space inside an element, pushing the content away from the edges, or shrink or increase the inside of the element to a degree. To customise the padding of an element using CSS, you can type:
padding: INSERT VALUE HERE;

If you want to be much more specific on exactly which area of the padding that you want to customise you can type a dash and a direction (top, left, right, and bottom) after the padding to specify which padding you want to change. For example:
padding-left: 8px;

Diagram showing where the padding is in an element

Border

In HTML, a border refers to the boundary or outline that surrounds an element. It is a visual property that helps differentiate and define the edges of an element on a web page. Borders can be applied to various HTML elements such as <div>, <p>, <h1>, and more. They can be customized in terms of color, style, and width.

Diagram showing where the border is in an element

Margins

Much like in English, the margin is basically the space between where the element originally was and where it currently is. This might sound confusing at first, but when you first place a element on a page, there is going to be a default placement for the element on the page. If you use a declaration such as margin-left: 20px;, then the element will shift from its original position to 20px to the right of its original position.

To customise the margin of the element, you can use the declaration:
margin: top Spacing right Spacing bottom Spacing left Spacing;
Alternativly, much like the padding declaration, you can add a dash and then type in the direction of the margin (top, bottom, left, right) to specify exactly where you want to place a margin spacing on your element.
